# ET18 書籍模組（操作手冊）— 已廢棄

> **狀態**：**廢棄（2026-04-24）**
>
> 原用 Moodle Book 模組作為線上操作手冊，但 Moodle Book 不支援外部程式化寫入（無 create_chapter 等 Web Service）；將手冊全放進 Moodle 也不便於主系統直接呼叫。
>
> **新方案**：改由獨立 **DM 模組**提供線上操作手冊，主系統各畫面以 func_name 直接呼叫 DM。
>
> **後續**：本檔保留作為歷史決策記錄，不再更新。
