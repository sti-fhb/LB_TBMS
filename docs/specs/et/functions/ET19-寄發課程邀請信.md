# ET19 寄發課程邀請信

## Moodle 操作路徑

- Moodle Plugin: **enrol_invitation**（Invitation enrolment）
- 課程 →「Participants」→「Enrolment methods」→「Invitation」→「Send invitation」

## 功能說明

輸入學員 eMail → 系統自動組合邀請信（含課程名稱、邀請連結、邀請碼）→ 透過 SMTP 寄出。
