# UCDM001-文件管理與版本管控

資訊人員於 DM 文件管理系統上傳、更新作業規範、SOP 文件、訓練教材，系統自動管理版本歷程並提供電子簽核流程。

- **主要參與者**：資訊人員
- **前置條件**：已登入 DM（透過 DP Token）且具文件管理權限
- **後置條件**：文件已發布，使用者可透過 Moodle 連結或主系統功能呼叫查閱最新版

## 正常流程

1. 進入 DM 文件管理頁面
2. 選擇文件分類（SOP / 手冊 / 教材 等）
3. 上傳新版文件（PDF/Word 或線上編輯）
4. 填寫版本備註（變更摘要）、版本號（v1.0 / v2.0...）
5. 送交簽核，依 sopRole 工作流（SOP_AUTHOR → SOP_REVIEWER）
6. 核准後系統建立新版本，保留舊版本歷程
7. 自動通知訂閱者 / 相關角色

## 替代流程

- **3a**. 線上編輯器撰寫（HTML / Markdown）
- **6a**. 退件回擬稿者修正
- **7a**. 可回溯查看 / 下載 / 還原歷史版本
