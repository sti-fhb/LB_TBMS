# UCET012-查閱線上操作手冊

使用者在任何操作畫面按 [F1] 開啟對應的線上操作手冊。系統以 func_name（功能代號）自動對應手冊檔案。

- **主要參與者**：使用者
- **前置條件**：已登入；資訊人員已將操作手冊（PDF）上傳至指定位置
- **後置條件**：無（純查詢）

## 正常流程

1. 使用者在操作畫面按 [F1]
2. 系統依目前畫面的 func_name 查詢對應操作手冊
3. 開啟並顯示操作手冊內容
4. 使用者線上瀏覽或下載

## 替代流程

- **2a**. 該功能尚無操作手冊，顯示「尚無操作手冊」提示

## 前置作業（資訊人員）

- 將各功能的操作手冊（PDF）放置到指定目錄（以 func_name 為檔名對應，如 `IFSM01-系統登入頁.pdf`）
