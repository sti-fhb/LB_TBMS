# 教育訓練模組 需求清單（RQET）

> 本文件以 EA Model 之 UseCase 結構及主專案初步分析資料（`_refs/10-教育訓練.md`）為主體，對應需求說明書之 RQ 編號。
> 無對應 RQ 的項目以「—」表示，來源標註為「分析資料」。
> 本模組採用 **Moodle 4.5 LTS** 作為教育訓練平台。
>
> **編碼原則（2026-04-24 確認）：** RQ 編碼綁 RFP 章節。RQDP041~043（RFP 1.7 文件管理與教育訓練，屬基礎架構章節）編碼**保留於 DP 模組**，功能由 ET 模組實作承接，本文件以 cross-reference 方式列出。文件管理（DM）為獨立模組，RQDM* 不納入 RQET（另見 `RQDM.md`）。

## 模組概述

教育訓練模組提供血液中心內部人員的線上教育訓練與知識管理功能，包含課程管理、線上測驗、學員管理、SOP 文件協作編輯、操作手冊線上查閱。本模組為周邊支援系統，不參與血品主流程，採 Moodle LMS 作為平台。

### 操作角色

| 角色 | 職責範圍 |
|------|----------|
| 管理者（講師） | 建立課程、上傳教材、編排章節、建立測驗、邀請學員、檢視學習報表 |
| 學員 | 加入課程、學習教材內容、進行線上測驗、查看個人學習進度 |
| 資訊人員 | 維護 SOP 文件（Wiki 模組）、上傳操作手冊（Book 模組）至 Moodle |
| 使用者 | 系統一般使用者，透過 [F1] 查閱線上操作手冊 |
| eMail Server | 外部郵件系統，負責寄發課程邀請碼通知信 |

### 作業流程

```
課程管理（管理者作業）
──────────────────
1. 管理者建立新課程（名稱、描述、分類、封面圖片）
2. 上傳教材（影片、PDF、PPT、圖片或混合媒材頁面）
3. 編排課程章節順序，設定進度條件（強制觀看完成/測驗通過）
4. 建立線上測驗（題庫管理、單選/多選、自動閱卷）
5. 邀請學員加入課程（Email 邀請信/邀請碼/手動加入）
6. 檢視學習報表與完課率

學員學習作業
──────────
7. 學員從 Email 取得邀請碼，輸入邀請碼或點選邀請連結加入課程
8. 依序學習課程章節（影片、文件、圖文），系統追蹤進度
9. 完成章節後進行隨堂測驗，系統自動閱卷
10. 查看個人學習進度與測驗成績

SOP 文件管理（資訊人員作業）
──────────────────────────
11. 資訊人員透過 Moodle Wiki 模組協作編輯 SOP 文件，系統保留版本歷程

操作手冊查閱（使用者作業）
────────────────────────
12. 使用者在操作畫面按 [F1]，系統依 func_name 開啟對應 Moodle Book（操作手冊）
```

## 跨模組介接

| 介接方向 | 介接對象 | 用途 |

| 資訊模組（DP） → 教育訓練 | 資訊模組（DP） | 使用者帳號由資訊模組統一管理，Moodle 透過 SSO（SAML/OAuth）與系統身分整合 |

| 各業務模組 → 教育訓練 | 全系統 | 使用者在各模組操作畫面按 [F1] 觸發 ET 的操作手冊查閱功能 |
